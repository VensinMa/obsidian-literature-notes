## Tag 命名规范

1. 使用中文（除非是专有名词）
2. 不超过 10 个字符
3. 避免重复含义
4. 保持一致性

## Tag 数量限制

### Zotero 限制
- **单个条目**：无官方限制，实测 50+ 个 tags 正常工作
- **建议数量**：5-15 个 tags 为宜
- **过多 tags 的问题**：
  - 影响检索效率
  - 增加维护成本
  - 降低标签的区分度

### 推荐配置
- **最少**：3 个（文档类型 + 研究领域 + 物种）
- **推荐**：5-8 个（覆盖主要维度）
- **最多**：15 个（包含所有相关标签）

### Tag 维度分配
| 维度 | 数量 | 说明 |
|------|------|------|
| 文档类型 | 1 | literature/review/preprint |
| 研究领域 | 1-2 | 基因组学、群体遗传学等 |
| 技术方法 | 1-2 | 泛基因组、T2T组装等 |
| 特征元素 | 1-2 | TE、着丝粒、卫星DNA等 |
| 物种 | 1 | 人类、水稻、苹果等 |
| 应用 | 0-1 | 注释工具、变异检测等 |
| 专有名词 | 0-2 | EDTA、RepeatMasker等 |

### 示例
```
# 最小配置（3 个）
tags: [literature, 基因组学, 橡树]

# 推荐配置（6 个）
tags: [literature, 基因组学, 泛基因组, 橡树, 局部适应, 渗入]

# 完整配置（10 个）
tags: [literature, 基因组学, 泛基因组, 群体遗传学, 植物, 橡树, 结构变异, 渗入, 局部适应, ERF基因]
```
