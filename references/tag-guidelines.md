# Tag 规范指南

## Tag 层级结构

### 1. 文档类型
- `literature` - 期刊论文
- `review` - 综述
- `preprint` - 预印本
- `method` - 方法论文
- `thesis` - 硕博论文

### 2. 研究领域
- `基因组学`
- `转录组学`
- `蛋白质组学`
- `表观遗传学`
- `群体遗传学`
- `进化生物学`
- `临床遗传学`
- `作物育种`
- `生物信息学`

### 3. 技术方法
- `泛基因组`
- `长读长测序`
- `Hi-C`
- `T2T组装`
- `k-mer`
- `机器学习`
- `深度学习`
- `比对`

### 4. 特征元素

## Tag 命名规范

1. 使用中文（除非是专有名词）
2. 不超过 10 个字符
3. 避免重复含义
4. 保持一致性

## Tag 数量要求

- **最少**：10 个 tags
- **推荐**：15-20 个 tags
- **原则**：足够详细，确保文献可被准确检索

### 必选维度（每个文献至少包含）
| 维度 | 最少数量 | 说明 |
|------|----------|------|
| 文档类型 | 1 | literature/thesis/preprint |
| 研究领域 | 2-3 | 基因组学、群体遗传学等 |
| 技术方法 | 2-3 | 泛基因组、T2T组装等 |
| 特征元素 | 2-3 | TE、着丝粒、卫星DNA等 |
| 物种 | 1-2 | 人类、水稻、苹果等 |
| 应用 | 1-2 | 注释工具、变异检测等 |
| 专有名词 | 2-4 | EDTA、RepeatMasker等 |
